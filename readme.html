Lección 1

¿Qué es un README?

El README es la portada técnica del proyecto. Es el primer documento que verá la mayoría de las personas cuando accedan al repositorio.

No es

Una simple introducción o un archivo decorativo.

Es

El documento que permite comprender el proyecto completo.

Lección 2

Preguntas que debe responder

Todo README debería responder rápidamente las siguientes preguntas:

  • ¿Qué es?
  • ¿Para qué sirve?
  • ¿Cómo funciona?
  • ¿Cómo se fabrica?
  • ¿Cómo se prueba?
  • ¿Dónde está la documentación?
